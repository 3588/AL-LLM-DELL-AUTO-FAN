# AL-LLM-DELL-AUTO-FAN
自动调节DELL服务器风扇

如果温度低于60, 10%风扇

如果60-70 50%风扇

高于70 改为自动

## 准备工作
- 打开iDRAC
- 网络口连接上iDRAC口
- 下载openhardwaremonitor 然后 打开远程访问
- 建议先运行bat处理文件,确保测试通过
- py fan.py
